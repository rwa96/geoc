◆ update_loc()

void update_loc ( context *  ct)

Obtain current location from GPS module's serial connection and calculates distance to current goal.

  • If its not possible to obtain location data (see GPS_TIMEOUT) satcon wil be set to false (state::satcon)
  • If no goals have been fetched (#context::st::fetched) this function will just update satellite connection status but not the distance.
Parameters
ct(to check and update context::distance and context::st)
